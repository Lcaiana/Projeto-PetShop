# Projeto-PetShop
Primeiro projeto feito em C# no Visual Studio

##📌 Pré-requisitos para rodar:
<br>
1. Visual Studio instalado (com suporte a Windows Forms / .NET Desktop Development). <br>
2. Wamp64, XAMPP ou MySQL Server instalado localmente.
<br>
<br>
##🚀 Como executar o projeto:
<br>
1. Configurar o banco de dados: <br>
   *Certifique-se de que o Wamp64/MySql está ativo. <br>
   *Acesse o banco com o usuário 'root' e sem senha (de forma padrão). <br>
   *Importe o arquivo de script do banco de dados (.sql).<br> <br>
2. Abrir a Solução:
   *Clone ou baixe o repositório. <br>
   *Abra o arquivo `ProjetoPetShop.sln` no Visual Studio. <br> <br>
3. Rodar:
   *Pressione `F5` ou clique em **Iniciar** no Visual Studio para compilar e rodar o programa.
